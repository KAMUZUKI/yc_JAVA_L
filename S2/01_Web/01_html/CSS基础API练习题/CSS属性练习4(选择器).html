CSS选择器

页面
标签选择器
又称为元素选择器 ，最为基本 ，格式：标签名{}
ID选择器
根据id选择标签 ，格式：#id{}
Class选择器
根据class名，设置的选择器 ，格式：.class名{}
并集选择器
创建由逗号分隔的多个选择器 ，格式：选择器，选择器，选择器{}
后代选择器
多个选择器以空格分开，组合成包含关系 ，格式：父选择器名   子代选择器名{}
子元素选择器
选择匹配元素中的直接后代目标元素 ，语法：第一个选择器>第二个选择器
兄弟选择器
相邻兄弟选择器可以选择紧接着另一元素后的元素，必须得是同一个父元素 ，格式：a+b{} 
伪类选择器
根据条件的改变匹配元素，是相对于文档的固定状态来说的。
属性选择器
属性选择器:对带有指定属性的 HTML 元素 ，格式：标签名[属性名='值']{}
